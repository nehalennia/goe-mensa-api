Endpoints

  • GET /mensen - returns a list of mensen/canteens
  • GET /categories - returns a list of dish categories (parameters: mensa)
  • GET /dishes - returns a list of dishes (parameters: date, mensa, category)
  • GET /additives - returns a list of additives (parameters: ids)
  • GET /priceTypes - returns a list of price types
